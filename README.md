## 📌 Project Overview

This Flutter project is a **Digital ID Card application** that displays personal and academic information in a clean and modern UI using Material Design components.

The app demonstrates structured layout design using Flutter widgets like **Card, ListTile, CircleAvatar, and Divider**.

---

## 🎯 Features

* ✅ AppBar with centered title
* ✅ Card layout for ID design
* ✅ Profile image using CircleAvatar
* ✅ Name and department display
* ✅ Divider for section separation
* ✅ ListTile for structured information
* ✅ Clean and modern UI

